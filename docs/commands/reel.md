# Reel Commands

This documentation is currently under heavy development.

## Usage

Call a Reel Command

```shell
reel <command>
```

Call Help for a Reel Command

```shell
reel <command> --help
```

## Commands

### properties

Show all evaluated and compiled available properties

```shell
reel properties
```

### docker:build

Building Docker Container

```shell
reel docker:build
```

### docker:push

Pushing Docker Container(s)

```shell
reel docker:build
```
